# Marketing Homepage Hero Example

This example shows how to use the scaffold for a realistic first task.

## Scenario

A marketing homepage hero section must be implemented from a high-fidelity design.
The main risk is that a coding agent turns it into a generic SaaS hero with default typography, weak spacing, and a standard button style.

## Goal

Use the implementation spec and component map to preserve:

- strong headline hierarchy
- clear value proposition structure
- premium but restrained styling
- a truly primary CTA
- responsive behavior that does not flatten the layout

## Files

- `implementation-spec.yaml`
- `component-map.json`
- `acceptance-checklist.md`

Use this example as a reference when creating your own first task.
